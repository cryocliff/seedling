# seedling documentation

**seedling** puts everything Python — interpreters, isolated environments, a
portable [VS Code](https://code.visualstudio.com/), cloned repos, config, and
logs — in one tidy folder, set up for you in a single command. For the
quickstart and the "why", see the
[project README on GitHub](https://github.com/cryocliff/seedling).

These pages are the full reference:

- **[Full documentation](DOCUMENTATION.md)** — every command and flag, the
  folder layout, the update model, organization/multi-user deployment, and
  troubleshooting.
- **[Offline / air-gapped installs](OFFLINE.md)** — running with no internet
  at all, from a network share or private mirror.
